# Springboot-learn
主要是用来记录学习spring-boot时所作的一些demo
## firstdemo
第一个入门小程序 -> helloWorld
## customPort
自定义springboot端口号
## indexPage
跳转到静态页面的demo
## thymeleaf
thymeleaf模板使用demo
## exception
异常处理的demo
## jdbcMysql
通过jdbc操作数据库
## aspect
使用aop来记录日志
## asyncScheduled
使用异步方式进行调度
## log4j2
使用log4j2进行日志记录
## properties
使用配置文件设置一些常量,如jdbc连接信息等

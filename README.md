# Springboot-learn
主要是用来记录学习spring-boot时所作的一些demo

# 简要说明
此专栏中所有的demo都依赖parent里面的pom.xml
如果您不希望下载所有的demo,你可以将子模块依赖父模块的pom删掉,并在子模块中添加父模块部分的依赖即可
